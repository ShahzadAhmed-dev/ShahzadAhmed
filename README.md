# ShahzadAhmed

A personal portfolio website showcasing my journey as a Mathematics instructor, content writer, and BS Software Engineering student at the Virtual University of Pakistan. This portfolio highlights my passion for education and technology, combining teaching experience with growing expertise in software engineering.

## 🚀 Features
- **About Me** – Background and professional journey
- **Skills** – Technical and academic strengths
- **Projects** – Selected coding and writing projects
- **Blog** – Articles on technology, education, and continuous learning
- **Certifications** – Achievements and credentials
- **Contact** – Easy ways to connect

## 🛠 Tech Stack
- HTML5  
- CSS3  
- JavaScript (basic)
